Suggested Technology¶

  1. Build engine: Cmake
  2. C++ template library for numerics: Eigen
  3. Scientific computing: Numpy
  4. Static compiler: Cython
  5. Figures: Bokeh
  6. Helper: Eigency
  7. Command line library: Click
