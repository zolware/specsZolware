The Computational Tool (API)¶

Zolware Inc. provides an open source computational tool package written in Python (with C++ extensions when necessary). The packages is freely available on the Python Package Index (PyPI) and on a GitHub repository. The API consists in tools to process, analyse, and control data.

From its open source nature, the API makes an heavy used of third-party scientific packages such as numpy, scipy, and Eigen (C++).
