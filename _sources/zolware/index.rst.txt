===================
Zolware's Products
===================

Zolware Inc. propose three different services:

1. A computational tool. A free open source API to process, analyse, and visualize data.
2. A web service. A pay-per-use web service that allows registered users to define, stream, and update data. The service also uses the free open source API for data analysis and vizualsation and provides a digital market palce were registered users can buy, sell and share their data.
3. A customized service with dedicated solutions in forms of web applications.
